# Start Your Cyber Security Journey Lab

## 1. Project Overview
This project summarizes the core concepts from TryHackMe's Start Your Cyber Security Journey module. The goal is to explain the difference between offensive security, defensive security, and search skills used by cybersecurity beginners.

## 2. Offensive Security
Offensive security focuses on finding weaknesses before real attackers do. An ethical hacker tests systems legally and safely with permission.

Key ideas:
- Think like an attacker
- Find vulnerabilities
- Test safely
- Stay within scope
- Report findings responsibly

## 3. Defensive Security
Defensive security focuses on protecting systems and responding to attacks. Defenders monitor systems, investigate alerts, patch weaknesses, and reduce risk.

Key ideas:
- Protect systems
- Monitor activity
- Detect suspicious behavior
- Respond to incidents
- Improve security controls

## 4. Offensive vs Defensive Security
| Area | Offensive Security | Defensive Security |
|---|---|---|
| Goal | Find weaknesses | Protect systems |
| Mindset | Think like an attacker | Think like a defender |
| Example | Test a vulnerable web app | Block or investigate an attack |
| Output | Findings report | Alerts, fixes, and response actions |

## 5. Search Skills
Search skills are important because cybersecurity requires constant research.

Security professionals use search to find:
- Documentation
- Error messages
- Vulnerability information
- Tool usage examples
- Security concepts
- CVEs and advisories
- Proof of concepts

## 6. Useful Search Techniques
| Technique | Example |
|---|---|
| Exact phrase search | `"failed login attempts"` |
| Search a specific site | `site:docs.python.org sqlite3` |
| Exclude a word | `python sqlite -django` |
| Search file types | `filetype:pdf cybersecurity basics` |
| Combine keywords | `linux permissions chmod examples` |
