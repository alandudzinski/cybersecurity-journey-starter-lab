# Cybersecurity Search Skills Cheat Sheet

## Search Operators
| Goal | Search Example |
|---|---|
| Find exact phrase | `"SQL injection prevention"` |
| Search one website | `site:owasp.org SQL injection` |
| Exclude results | `linux permissions -windows` |
| Find PDFs | `filetype:pdf cyber security basics` |
| Find documentation | `python sqlite3 documentation` |

## Cybersecurity Research Examples
| Task | Search Query |
|---|---|
| Learn a concept | `what is phishing cybersecurity` |
| Understand an error | `"permission denied" linux chmod` |
| Find official documentation | `site:docs.python.org csv module` |
| Research a vulnerability | `CVE-2024 example vulnerability` |
| Learn a defense | `how to prevent brute force login attacks` |

## Research Rules
- Prefer official documentation first.
- Check the date of security information.
- Avoid blindly copying commands.
- Understand what a command does before running it.
- Never test attacks on systems you do not own or have permission to test.
